  1. Introduction

    Weather Information Management System [WIMS] [VERSION NUMBER] is the latest version. WIMS is a self contained .jar executable
    program that fetches real time weather data from the Bureau of Meteorolgy website and displays it in a user friendly format.
    The program is written in Java [VERSION] and requires your operating system to have Java [VERSION] installed.

  2. Installation

    1. Installing Java

      2. To install Java head to the Java website and download the latest version. Follow it's installation instructions.

    2. Installing Weather Information Management System

      3. WIMS comes as a self contained .jar executable and does not require installation in order to launch.

  3. Launching the program

    4. To launch WIMS simply double click the .jar file, or right click and select "OPEN".

  4. Quick Overview


    5. Upon launch, WIMS should look similar to the above image. This screen contains all you need to begin navigating the program.

    1. Section 1 in the image above is the list of states weather data is provided for. Clicking any of these will open up a selection of weather stations to view.
    2. Section 2 in the image above is the map display. This is simply a graphical display to assist in visualising selecting a state from the list.
    3. Section 3 in the image above is the list of favourite weather stations stored. Any weather stations added to favourites will appear here.

  5. Checking the weather

    1. Choosing a state

      2. When the program is initially opened, a list of states is displayed on the left-hand side. (See box 1 for list of states)
      Simply hover or click on the state in order to reveal a panel containing that state's areas and weather stations. (See box 2 for example of a state)

    2. Finding your area

      3. Once you've chosen a state, simply scroll through the list of areas until you find your desired area. [ ! Perhaps I can implement alphabetical sorting?]

    3. Finding your weather station

      4. To find your weather station, simply scroll through the list of areas weather stations.

    4. Displaying weather data

      5. To display weather data for your desired weather station simply click on the weather station's name.


      The weather data for that weather station will appear in a table format similar to the one below.

      To display the data in a more visual form, simply click the down arrow to display a graph of the data.

  6. Favourite weather stations

    1. Adding a favourite weather station

      2. To add a weather station to your favourites list, simply open that weather station to view it's data
      and left click the button "+ Add Favourite"

    2. Removing a favourite weather station

      3. To remove a weather station from your favourites list, simply open that weather station to view it's data
      and left click the button "- Remove Favourite"

  7. Uninstall

    8. The program does not need a proper uninstallation to be removed, as it does not need to be installed. Instead, simply delete the .jar file and any assosicated files.
